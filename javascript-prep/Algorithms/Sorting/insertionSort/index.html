What is Insertion Sort?

Insertion Sort is a simple and intuitive sorting algorithm that builds the sorted array one element at a time. It works by repeatedly taking an element from the unsorted part of the array and inserting it into the correct position in the sorted part.

How It Works

  • Start with the first element, which is already sorted.
  • Pick the next element and compare it with elements in the sorted part.
  • Shift larger elements to the right to make space for the new element.
  • Insert the element into its correct position.
  • Repeat for all elements in the array.

Key Characteristics

  • In-Place Sorting: No extra memory is required apart from the input array.
  • Stable Sorting: Maintains the relative order of equal elements.
  • Best for Nearly Sorted Data: Performs well when the array is nearly sorted.

Time Complexity

  • Best Case: O(n) (when the array is already sorted).
  • Worst Case: O(n2) (when the array is sorted in reverse order).
  • Average Case: O(n2).

Space Complexity

  • Space: O(1) (only uses a constant amount of extra space).

JavaScript Implementation

Iterative Version

const insertionSort = (arr) => {
  for (let i = 1; i < arr.length; i++) {
    let current = arr[i];
    let j = i - 1;

    while (j >= 0 && arr[j] > current) {
      arr[j + 1] = arr[j];
      j--;
    }

    arr[j + 1] = current;
  }
  return arr;
};

console.log(insertionSort([4, 6, 1, 3, 8, 5, 2, 89, 55, 67]));
