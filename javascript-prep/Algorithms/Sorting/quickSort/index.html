Quick Sort

What is Quick Sort?

Quick Sort is a divide-and-conquer sorting algorithm that selects a pivot element, partitions the array around it, and recursively sorts the sub-arrays.

How It Works

  1. Choose a Pivot: Select a pivot element (commonly the last element of the array).
  2. Partition: Rearrange the array so that elements smaller than the pivot are on the left and elements greater than the pivot are on the right.
  3. Recursively Sort: Recursively apply the same logic to the left and right partitions.

Time Complexity

  • Best Case: O(n log n)
  • Worst Case: O(nÂ²) (when the pivot is always the smallest or largest element).
  • Average Case: O(n log n)

Space Complexity

Space: O(log n) due to recursion.

JavaScript Implementation

Quick Sort Algorithm


const quickSort = (arr) => {
  if (arr.length <= 1) return arr; // Base case for recursion

  const pivot = arr[arr.length - 1];
  let left = [];
  let right = [];
  
  for (let i = 0; i < arr.length - 1; i++) {
    if (arr[i] < pivot) left.push(arr[i]);
    else right.push(arr[i]);
  }

  return [...quickSort(left), pivot, ...quickSort(right)];
};

console.log(quickSort([4, 6, 1, 3, 8, 5, 2, 89, 55, 67]));

Example Output

[1, 2, 3, 4, 5, 6, 8, 55, 67, 89]

When is Quick Sort Useful?

  • Large Datasets: Ideal for handling large datasets efficiently.
  • In-Place Sorting: Quick Sort is an in-place algorithm.
  • Unstable Sort: Not stable, meaning equal elements' order is not preserved.
